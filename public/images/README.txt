
TRANQUIL MENTAL HEALTH - ASSET DIRECTORY
========================================

Place your clinical images, staff headshots, and logos in this directory.

HOW TO USE IN CODE:
-------------------
In your React components (e.g., HomeView.tsx), reference these images using a relative path:

Example:
<img src="public/images/doctor-portrait.jpg" alt="Dr. Smith" />

Recommended Image Specifications:
- Hero Images: 1920x1080px (WebP or Optimized JPG)
- Profile Photos: 400x400px (PNG or JPG)
- Logos: SVG (Scalable Vector Graphics)
